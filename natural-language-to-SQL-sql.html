2. 数据库的基本概念

SQL的介绍 

prices[1,2,3,4,5]
N=len(prices)
max_ij=(0,0)
max_num=prices[max_ij[1]]-prices[max_ij[0]]
for i in range(N):
    for j in range(N):
        if j>i:
            new_max_num=prices[j]-prices[i]
            if new_max_num>max_num:
                max_num=new_max_num
                max_ij=(i,j)
